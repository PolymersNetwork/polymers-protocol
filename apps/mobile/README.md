**Installation**

```bash

cd polymers-recycling-app
npm ci
cp .env.example .env
# Configure environment variables
npx expo start
```

**Build & OTA Deployment**

```bash
# iOS
npx eas build --platform ios
# Android
npx eas build --platform android
# OTA Updates
npx eas update
```
