```bash
git clone https://github.com/PolymersNetwork/polymers-recycling-app.git
cd polymers-recycling-app
npm ci
cp .env.example .env
# Configure environment variables
npx expo start
```

**Build & Deploy**

```bash
# iOS
npx eas build --platform ios
# Android (SMS & Seeker-optimized)
npx expo run:android
# OTA Updates
npx eas update
```
